# seba-fresh

## Chosen configurations

**General**

* Build Systems: `Gulp`
* Testing: 
  * `Jasmine`

**Client**

* Scripts: `Babel`
* Markup:  `HTML`
* Stylesheets: `Sass`
* Angular Routers: `ui-router`
* CSS Frameworks: `Bootstrap`
  * Option to include `UI Bootstrap`

**Server**

* Scripts: `Babel`
* Database:
  * `MongoDB`
    * Authentication boilerplate: `Yes`
    * oAuth integrations: `Facebook`
    * Socket.io integration: `Yes`
